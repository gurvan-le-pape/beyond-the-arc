# Icon Usage Guidelines

## Where to get icons

- Use https://heroicons.com/ for all new icons.
- Copy SVG of the icon you need.
- Open Notepad
- Paste SVG code
- Save icon.svg (File type: All Files)

## Where to place icons

- Place the SVG file in `public/images/icons/`.
- Name the file with a clear, lowercase name (e.g., `mail.svg`, `phone.svg`, `link.svg`).

## How to use in code

- Reference the icon in your code using `/images/icons/<iconname>.svg`.
- Example:
  ```html
  <img src="/images/icons/mail.svg" alt="mail" />
  ```

## Why Heroicons?

- Consistent, modern, open-source icons.
- Easy to customize and style.

---

**Always use Heroicons for new icons to keep the UI consistent!**
